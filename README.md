# Vercel Backend Test

This is a minimal backend for testing API routes on Vercel.

## Deploy Steps
1. Push this folder to GitHub.
2. Go to https://vercel.com/new
3. Select your GitHub repo.
4. Deploy.

After deployment, you will get URLs like:

- https://yourproject.vercel.app/api/start-session
- https://yourproject.vercel.app/api/chat
- https://yourproject.vercel.app/api/upload-file
